# SmartCalcAI Bot – The Ultimate Telegram Math Assistant

A feature-rich Telegram bot that provides step-by-step solutions to mathematical problems, from basic arithmetic to differential equations, Laplace transforms, vector calculus, and more. Users can bring their own OpenAI/Gemini/Claude API key for natural language understanding. Premium features are unlocked with Telegram Stars.

## Features

### Free Tier (10 calculations/day)
- Derivatives, integrals, limits, series expansions
- Ordinary differential equations (ODEs)
- Laplace & Fourier transforms
- Gradient, divergence, curl (vector calculus)
- Numerical root finding, integration, minimization
- 2D plotting
- Matrix operations (multiply, inverse, determinant)
- Unit conversion
- Statistics (basic stats, t-test, linear regression)
- Calculation history

### Premium Features (Telegram Stars)
- 3D surface plots
- System of equations solver
- Curve fitting
- PDF export of calculations
- Share calculations with others
- Save custom functions
- No daily limits
- Priority processing

### Bring Your Own LLM
- Securely store your OpenAI/Gemini/Claude API key (encrypted)
- Send natural language queries – the bot interprets and suggests the right command
